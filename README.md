# Nil Bangoriya — AI/ML Engineer Portfolio

A stunning, unique portfolio website built to showcase AI/ML engineering skills and land your next role.

## Features

- Interactive neural network canvas background
- Custom cursor with hover effects (desktop)
- Typing animation cycling through ML keywords
- Glassmorphism UI with cyan/purple/magenta gradient theme
- Fully responsive with mobile navigation
- Scroll-triggered animations via Framer Motion
- Single config file for all personal content

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Customize Your Info

Edit **`src/data/profile.ts`** — this is the only file you need to update:

- Name, title, tagline, location
- Email, GitHub, LinkedIn links
- About paragraphs and stats
- Skills, projects, and experience
- Hero typing words

## Build for Production

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to [Vercel](https://vercel.com), [Netlify](https://netlify.com), or GitHub Pages.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
